ignore-cmd
==========
add paths to .gitignore .npmignore from cmd

```sh
Usage:    ignore <flag> <entry> [...entries] 

    flag      -g  .gitignore
              -n  .npmignore
              -b  both

    entry      The path you want to ignore (e.g. /node_modules/)

    entries    Ignore multiple entries
```
